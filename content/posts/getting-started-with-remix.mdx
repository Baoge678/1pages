# 开始使用 Remix 构建现代 Web 应用

Remix 是一个全栈 Web 框架，它让你专注于用户界面，并通过 Web 基础来部署你的应用程序。本文将带你了解 Remix 的基础知识。

## 为什么选择 Remix?

Remix 提供了许多现代 Web 开发所需的功能：

- 服务器端渲染 (SSR)
- 嵌套路由
- 并行数据加载
- 优秀的错误处理
